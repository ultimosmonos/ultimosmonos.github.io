Últimos monos

The Últimos monos1 blog is meant to be a blank canvas, the place where I’ll be describing the kind of tasks I have to do for a living and any other I do just for fun.

This blog will be my own Knowledge-Base, open to the world just in case it can help someone else.

Behind the Blog

I’ve chosen to use Jekyll side by side with the GitHub Pages; a combination too tempting to ignore and open another Wordpress blog. I’ve didn’t use these technologies before, so I’ll do my best to learn how to do things well.

The first thing I’ve done was to search for a cool Jekyll theme and I’ve found HPSTR, which I liked a lot. After some tweaking to personalize it a bit and to add multi language support (thanks to this post from Sylvain Durand) this is what I’ve got.

I like it.

Diego

A guy that seems to enjoy making things difficult, avoiding the shortcuts. When there is a problem with two possible solutions, one of them is simple, easy to implement and almost Plug&Play, and the other is slightly more sophisticated which undoubtedly will require more effort and some learning process, the balance usually tilts to the latter option. This blog is just an example.
